# 金金管家 - 启动说明

## 🚀 快速启动

### 方法一：一键启动（推荐）

在项目根目录打开终端，运行：

```bash
npm run dev
```

这个命令会：
1. 自动启动 Vite 开发服务器（React 应用）
2. 自动编译 Electron 主进程
3. 等待服务器启动后自动打开 Electron 窗口

### 方法二：分步启动

如果你想分别控制每个进程：

**步骤 1：启动 Vite 开发服务器**
```bash
npm run dev:react
```

**步骤 2：等待服务器启动后（看到 "Local: http://localhost:5173"），在新终端运行：**
```bash
npm run build:electron
npx electron .
```

## 📝 使用说明

### 创建任务
- 在输入框中输入任务内容
- 按 `Enter` 键添加任务

### 管理任务
- **完成任务**：点击任务前的复选框 ✓
- **删除任务**：点击任务右侧的 × 按钮

### 窗口控制
- **置顶/取消置顶**：点击顶部工具栏的 📌 按钮
- **调整透明度**：拖动透明度滑块（30%-100%）
- **拖动窗口**：点击窗口内容区域拖动
- **最小化**：点击 `−` 按钮
- **关闭**：点击 `×` 按钮

## 🛠️ 故障排除

### 如果窗口是空白的
1. 确保 Vite 服务器正在运行（端口 5173）
2. 按 `F12` 打开开发者工具查看错误
3. 检查控制台是否有错误信息

### 如果 Electron 启动失败
```bash
# 重新安装 Electron
npm install electron --save-dev
```

### 如果端口被占用
修改 `vite.config.ts` 中的端口号，或停止占用 5173 端口的进程

## 📦 构建生产版本

```bash
npm run build
```

这会生成可执行文件在 `release` 目录中。
