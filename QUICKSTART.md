# 快速启动指南

## 第一步：安装依赖

```bash
npm install
```

## 第二步：启动开发环境

```bash
npm run dev
```

这将：
1. 启动 Vite 开发服务器（React 应用）
2. 编译 Electron 主进程代码
3. 启动 Electron 应用窗口

## 使用说明

### 创建任务
- 在输入框中输入任务内容
- 按 `Enter` 键添加任务

### 管理任务
- **完成任务**：点击任务前的复选框
- **删除任务**：点击任务右侧的 `×` 按钮

### 窗口控制
- **置顶/取消置顶**：点击顶部工具栏的 📌 按钮
- **调整透明度**：使用顶部工具栏的透明度滑块（30%-100%）
- **最小化**：点击 `−` 按钮
- **关闭**：点击 `×` 按钮

### 拖动窗口
- 点击窗口任意空白区域（非按钮区域）可以拖动窗口

## 功能特性

✅ **桌面固定清单** - 窗口可始终置顶，不会被其他窗口遮挡  
✅ **透明窗口** - 可调整透明度，不遮挡视线  
✅ **本地存储** - 任务数据自动保存，关闭后重新打开仍然保留  
✅ **美观UI** - 渐变背景、流畅动画、现代化设计  

## 故障排除

### 如果遇到 TypeScript 错误
这些错误通常在安装依赖后会自动解决。确保已运行 `npm install`。

### 如果 Electron 窗口无法打开
1. 确保 Vite 开发服务器已启动（http://localhost:5173）
2. 检查控制台是否有错误信息
3. 尝试重新运行 `npm run dev`

### 如果窗口无法拖动
- 确保点击的是窗口内容区域（非按钮区域）
- 窗口标题栏和控制按钮区域不可拖动（这是设计如此）
